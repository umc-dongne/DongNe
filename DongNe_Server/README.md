# 테스트 api

## 1. GET

```jsx
*/***
** API No. 0.1*
** API Name : GET 테스트 API*
** [GET] /test*
**/*
```

## 2. POST

```jsx
/**
 * API No. 0.2
 * API Name : POST 테스트 API
 * [POST] /test
 */
```

## 3. DB

```jsx
/**
 * API No. 0.3
 * API Name : db 테스트 API
 * [GET] /test/db
 */
```

**db 테스트 시, .env 파일을 별도로 생성하여, 해당 key값을 따로 적어야 함.**
