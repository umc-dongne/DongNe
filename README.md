## 동네의 기능을 소개합니다!

**동네**는 동아리 대표 계정인 단체 회원과 개인 계정인 일반 회원으로 이루어져 있습니다.

**단체 회원**은 동아리에 가입한 일반 회원의 명단과 출결을 관리하고, 회계 관리와 커뮤니티 이용이 가능합니다. 

**일반 회원**은 개인이 가입한 동아리의 기본 정보와 회계 현황을 조회하고, 본인의 출결 현황 조회와 처리, 커뮤니티 이용이 가능합니다.
